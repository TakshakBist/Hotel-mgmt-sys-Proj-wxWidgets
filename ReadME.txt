This will be a hotel management project using wxWidget


-----------------------------------------------------------------------------------------------------  
-----------------------------------------------------------------------------------------------------  

  Features:
  
  -> Check-in
  
  -> Check-out
       
  -> Search Guests
        
  -> Order Food
        
----------------------------------------------------------------------------------------------------  
----------------------------------------------------------------------------------------------------  

  Build Guide:

  This program is made using Microsoft Visual Studio 2019Community Edition and wxWidgets 3.1.5
    
  Install VS from its official website and wxWidgets from its official site:
      
  Here are the links:
  
  Visual Studio: https://visualstudio.microsoft.com/
       
  wxWdgets: https://www.wxwidgets.org/

   --------------------------------------------------------------------------------------------------  
    --------------------------------------------------------------------------------------------------  

  
    How to run:
 
    I recommend you to follow this video:    https://www.youtube.com/watch?v=FOIbK4bJKS8
       
    Follow the following steps for my  build:
  
    	1. Download VS Community Edition from : https://visualstudio.microsoft.com/
    
    
   	2.Download wxWidgets from : https://www.wxwidgets.org/
    
    
    	3.Extract the zip file of wxWidgets
    
    
    	4.Go to the build folder
    
    
 	5.Open w16.sln
    
    
	6.Batch build the solution
    
    
	7.close Visual studio
      
      
	8.Copy the path of the folder which contains wxWidgets
      
      
 	9.Add it to Environment variables and name it WXWIN
	
	
	10. Open the solution file and run it.
